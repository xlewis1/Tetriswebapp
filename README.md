# Tetris Web App

A classic Tetris game built with **HTML, CSS, and JavaScript**.  
Fully playable on **desktop, tablet, and mobile devices**, with responsive controls including a **D-Pad, rotate, hard drop, and hold**.

---

## Features

- Classic Tetris mechanics
  - 7-bag randomizer
  - Soft drop / hard drop
  - Rotation
  - Hold/Swap piece
- Scoreboard showing **Score, Level, Lines, and Held piece**
- Touch-friendly controls for mobile
- Keyboard support for desktop
  - Arrow keys: move / soft drop / rotate  
  - Space: hard drop  
  - Shift: hold piece
- Responsive design

---

## Files

- `index.html` — main HTML file
- `style.css` — styling
- `script.js` — game logic

---

## How to Play

### Desktop
- Move: Arrow Left / Right  
- Soft Drop: Arrow Down  
- Rotate: Arrow Up  
- Hard Drop: Space  
- Hold / Swap: Shift

### Mobile
- Use the on-screen **D-Pad** and buttons below the game canvas:
  - Rotate: ⟳  
  - Hard Drop: ⤓  
  - Hold: ⧉
